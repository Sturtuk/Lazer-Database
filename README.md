KDb - database based on JSON files
=============

PHP Library to use JSON files like a database.   


Requirements
-------
- Laravel 5.5
- Composer

Installation
-------

Add the following to your  `composer.json`  and run  `composer update` .

    "require": {
        "sturt/Kdb": "1.0.*"
    }
